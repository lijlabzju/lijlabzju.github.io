Welcome to Li-Lab!

The Li-Lab at Sir Run Run Shaw Hospital, Zhejiang University is focused on understanding the transcriptomic and epigenomic mechanisms underlying cancer development, using cutting-edge multi-omic methods and organoid system.

Single-cell multi-omic method

We use cutting-edge multi-omic sequencing technologies, or develop new methodologies if needed, to tackle important cancer questions!

Cell Research, 2017

Organoid system

We employ and optimize organoid system to model cancer development in vitro, which allows us to not only observe but also interfere the process!
